# EnergizedPower-Croptopia
Compatibility between Energized Power and Croptopia<br>
Download on CurseForge: [Energized Power - Croptopia](https://www.curseforge.com/minecraft/mc-mods/energized-power-croptopia)

## Information

**This mod requires [Energized Power](https://github.com/JDDev0/EnergizedPower) and [Croptopia](https://github.com/ExcessiveAmountsOfZombies/Croptopia/issues)**

**The fabric edition of this mod requires the Fabric API.**

*Feel free to include this mod in your ModPack :-)*
